1. FILE UPLOAD FOR QUESTION GENERATION:

On each study task modal, add:
- "Upload File" button (accept .pdf and .pptx files)
- For PDFs: Use PDF.js library to extract text
- For PowerPoints: Use mammoth.js or similar to extract text/content
- Send extracted content to Anthropic Claude API
- API call: POST to https://api.anthropic.com/v1/messages
- Prompt: "You are a Year 9/10 Victorian Curriculum tutor. Based on this content about [subject/topic from task], generate 8-10 study questions suitable for Year 9/10 students. Include: 3 multiple choice, 3 short answer, 2-4 extended response questions. Provide answers for all questions. Format as JSON."
- Parse API response and display questions in task modal
- Save questions to localStorage linked to that task
- "Regenerate" button for different questions
- Export questions as PDF

Upload UI:
- Drag-and-drop zone for files
- File type indicator (.pdf or .pptx)
- Upload progress bar
- "Generating questions..." loading state
- Preview uploaded file name
- Success message when questions generated
- Edit questions before saving
